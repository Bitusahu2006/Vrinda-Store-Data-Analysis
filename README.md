# Vrinda Store Data Analysis

A comprehensive data analysis project for Vrinda Store sales data with interactive Excel dashboard and detailed insights.

## 📊 Project Overview

This project involves analyzing Vrinda Store's sales data to understand business performance, customer behavior, and sales trends. The analysis includes data cleaning, exploratory data analysis, and the development of an interactive dashboard for data visualization and decision-making.

## ✅ What I Completed in This Project

### Data Cleaning & Preparation

- Cleaned and standardized the sales dataset.
- Removed inconsistencies in categorical fields such as Gender and Order Status.
- Formatted data types for dates, sales amounts, and percentages.

### Data Analysis

- Analyzed sales performance across months.
- Compared total sales and order volume using a combined chart.
- Identified the month with the highest sales and orders.
- Analyzed customer purchases by gender.
- Evaluated order status distribution (Delivered, Cancelled, Returned, Refunded).
- Identified the top-performing states based on sales.
- Studied the relationship between age group and gender based on order count.
- Determined the sales contribution of different sales channels.
- Identified the highest-selling product categories.

### Dashboard Development

- Built an interactive Excel dashboard using Pivot Tables and Pivot Charts.
- Added slicers for Month, Channel, and Category filtering.
- Created visualizations including:
  - Sales vs Orders Trend
  - Sales by Gender
  - Order Status Distribution
  - Top States by Sales
  - Orders by Age & Gender
  - Orders by Sales Channel
